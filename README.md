# NexusAI Analytics Dashboard

A fully responsive, dark-theme analytics dashboard inspired by the provided design reference.

## Tech Stack
- **HTML5** — Semantic structure
- **CSS3** — Custom properties (variables), animations, grid/flexbox layout
- **JavaScript (Vanilla)** — Dynamic calendar, dot-map canvas rendering
- **Chart.js 4.4** — All charts (sparklines, radar, donut, gauge, line, bar)

## Features
- KPI cards with live sparklines
- Interactive world dot-map with connection lines
- Radar chart for investment data
- Donut / gauge charts
- Multi-line trend chart
- Financial management bar charts
- Mini calendar with event highlights
- Schedule / agenda panel
- Animated entrance effects on all cards

| Font (body) | Rajdhani (Google Fonts) |

## How to Use
Just open `dashboard.html` in any modern browser — no build step required.
Internet connection needed only for Google Fonts and Chart.js CDN.
